    ‚≠ê 4. Function With Parameters

    Parameters = values we pass to a function.

    Example:

    A return sends a value back to the caller.

    print(result)



    ‚≠ê 6. TYPES OF PARAMETERS

    ‚úî 1. Positional Parameters

    Values are passed in correct order.

    def divide(a, b):
        print(a / b)
    divide(10, 2)  # correct


    ‚úî 2. Keyword Parameters

    You specify key=value.

    def info(name, age):
        print(name, age)
    info(age=21, name="Kumaran")

    ‚úî 3. Default Parameters

    If no value is passed ‚Üí default value used.

    def greet(name="Guest"):
        print("Hello", name)

    greet()
    greet("Kumaran")

    ‚úî 4. Variable-Length Parameters

    (a) *args ‚Üí accepts many values
    def total(*numbers):
        print(sum(numbers))

    total(1, 2, 3, 4, 5)

    (b) **kwargs ‚Üí accepts key=value pairs
    def details(**data):
        print(data)

    details(name="Kumaran", age=21)

‚≠ê 7. LAMBDA FUNCTION (Anonymous Function)

A lambda is a one-line function without name.

Syntax:

lambda arguments: expression

Example:

add = lambda a, b: a + b
print(add(4, 5))

More Lambda Examples:
‚úî Square of number
square = lambda x: x*x
print(square(6))

‚úî Even or Odd
is_even = lambda n: n % 2 == 0
print(is_even(10))

‚≠ê 8. RECURSION

A function calling itself.

Example: factorial using recursion

def fact(n):
    if n == 1:
        return 1
    return n * fact(n - 1)

print(fact(5))


Flow:

fact(5)
5 * fact(4)
4 * fact(3)
3 * fact(2)
2 * fact(1)
